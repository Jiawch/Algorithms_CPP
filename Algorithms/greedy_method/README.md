## 贪心算法
### 1 基本概念
1. 贪心算法的基本思路
 - 把待求解的问题分成若干个串行的子问题。
 - 对每个子问题的求解，是在一定的标准（限制条件）下，做一个当前看起来最好的决策，并且在以后的步骤中都不能再更改。（贪心策略）
 - 把子问题的解合并成原问题的解。

2. 特点
 - 贪心算法求解得到的解是局部最优解，不一定是全局最优解。
