### 回溯（DFS）

#### 写在前面

DFS是一个搜索算法，亦有模版可循，**在搜索的过程中，也可以顺便做点其他的工作**
```
step1: 画出类树图
step2: DFS
    DSF(节点/数组，参数):
        if 递归结束条件:
            return;
        for 枚举:
            DFS(节点/数组，参数) 
```
DFS是采用递归的写法，记得递归的三要素（递归参数和返回值，递归结束条件，递归公式）

#### 树DFS

- [树的先序/中序/后序/层次](./tree_order.cpp)

先序遍历：进入该点之前的操作  
后序遍历：离开该点之后的操作  
> 后序遍历的优点：后序遍历之后，可以对该点的数组、指针进行修改，因为该点已经访问过了
